# Graph2Heatmap

Graph Heatmap Visualization

## Requirement

Linux, OpenCV

## Usage

```bash
./tsv2heatmap --input=./twitter_rv.net --output=$HOME/twitter_rv.png --grid-width=16384 --brightness-increment=2
```
